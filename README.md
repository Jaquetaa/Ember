## Sobre o Projeto

Este projeto é uma construção de um drone com a adição de um comando para a nossa Prova de Aptidão Profissional (PAP), feita em C/C++ com ESP's e entre outros tipos de hardware, como sensores.

**O objetivo do drone é a detecção e prevenção de incêndios, e o comando vai pilotar o drone.**

Estamos a fazer este projeto usando o software [Arduino IDE](https://www.arduino.cc/en/software/) & [VSCode](https://code.visualstudio.com/), que depois salvamos no Github.
