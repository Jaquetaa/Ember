## Sobre o Projeto

Este projeto é uma construção de um drone para a nossa Prova de Aptidão Profissional feita em C/C++ com ESP's e entre outros tipos de hardware, como sensores.

O objetivo do drone é a detecção e prevenção de incêndios.
